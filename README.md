#### 使用条件

          运行本软件需要安装jdk6或以上版本JVM环境。

          双击ss.jar即可运行。

          本软件内置了自动升级功能，请勿更改文件的名称，否则可能无法使用。

#### 使用方法：

          1）右键可弹出菜单

2)快捷键 Alt+Q可快速隐藏显示

3)快捷键 Alt+E 退出软件。

          运行软件后，在运行目录会自动生成配置文件 ss.txt，可添加或者修改股票代码。"
